# fake-quotes-svc



### How to use

1) Run

    ```bash
    # create config file, change settings if you want
    cp config.yaml.dist config.yaml
    
    # build image accordingly your OS and processor architecture 
    make image/local
    
    # run local env
    docker-compose up 
    ```

2) Open http://localhost:18082/

3) Open browser console
